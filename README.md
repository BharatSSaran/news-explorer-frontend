# News Explorer

A React application that allows users to search for news articles using the News API and save them to their profiles.

## Features

- Search for news articles by keyword
- View search results in a clean, responsive interface
- User authentication and authorization
- Save and manage favorite articles
- Protected routes for authenticated users

## Technologies Used

- React
- React Router
- News API
- CSS3 with BEM methodology

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file with your News API key
4. Start the development server: `npm start`
